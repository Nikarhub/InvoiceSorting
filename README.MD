# PDF发票重命名工具
我姐是财务，每天需要重命名几百张发票，手动操作比较耗时，所以写了这个工具来自动重命名发票。
## 功能
自动扫描指定文件夹中的PDF发票文件，根据发票内容提取：
- 订单号(从备注里面获取)
- 发票号 
- 开票日期
并按格式重命名为：`{订单号}-{发票号}-{开票日期}.pdf`

## 安装依赖
```bash
pip install pdfplumber
```

## 使用说明
1. 将PDF发票放入`invoice`文件夹
2. 运行程序：
```bash 
python rename_invoice.py
```
3. 支持指定其他文件夹：
```bash
python rename_invoice.py
```
## 匹配规则
- 订单号：匹配"订单号/订单编号"等标签后的8位以上字母数字
- 发票号：匹配20位数字（自动去除空格）  
- 开票日期：支持多种格式，统一输出为"YYYY年M月D日"

## 日志说明
- `跳过（无法提取文本）`：PDF可能是图片型发票
- `跳过（缺少必要信息）`：会显示已识别到的字段
- `重命名成功`：显示旧文件名→新文件名

## 打包成exe
```bash
python -m PyInstaller -F -n "发票助手" rename_invoice.py
```
## 注意事项
1. 建议先备份原始文件
2. 遇到问题可查看终端输出的详细日志！
